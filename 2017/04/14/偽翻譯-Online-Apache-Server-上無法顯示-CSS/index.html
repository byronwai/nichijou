[偽翻譯] Online Apache Server 上無法顯示 CSS

前一段時間在弄 webpage design，用 Dreamweaver 弄好以後 Local 檢視完全未問題，可是放到 Server Side 就完全跑調了。仔細一看，發現 CSS 根本沒有套用上去啊。仔細 google 後，發現問題出在 CSS caching 上面。

0. 正文

我自己用的是 Byethost 跟 InfinityFree，結果發現 update 網頁後幾分鐘後，refresh 依然沒有變化，原來是 server side 為了節省資源而先把 style sheet 進行 caching。

1. 解決辦法

我們把以下的 code 放在 HTML 的 <> 之內，把 HTML 當作 PHP

或直接 include 在 PHP 之內
